This was a project for my EE2501 Digital Logic Design Final project with a partner. I assisted in making the code for both of the GAL22v10 PLD's in the circut using VHDL and converting it into a JEDEC file using ispLever, made the states diagram, and the circuit schemattic. Included in the repository is the VHDL
code, the state diagram, and the schematic for the circuit. PLD 1 acts as the main state machine for the traffic lights. PLD 2 provides the reset logic for the system.
